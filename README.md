Nilesh Sonavane Portfolio Website
Overview

This is a personal portfolio website built using:

Flask (Python)
HTML5
CSS3
Bootstrap 5
JavaScript

The portfolio showcases:

Personal introduction
Skills
Education
Certificates
Contact form with email integration

The project uses Flask template inheritance with a reusable base.html layout.

Features
Home Page
Introduction section
Profile image
Social media icons
Resume download button
Skills Page
Skill cards
Progress bars
Technology icons using Boxicons
Education Page
Educational qualification cards
Responsive Bootstrap layout
Certificates Page
Certificate showcase cards
Responsive image cards
Contact Page
Contact form
Gmail SMTP integration
Email directly sent to portfolio owner
Success popup alert
